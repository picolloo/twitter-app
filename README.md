# Twitter APP

This is a Twitter clone app to practice fullstack development.

## Backend

- Typescript
- TypeORM

## Functionalities

### MVP

- [x] Sign up
- [x] Login
- [x] User profile
- [x] Create tweet
- [x] Delete tweet
- [x] List tweets
- [x] Like tweet
- [x] List likes by user
- [x] List likes by tweet

### Plus

- [ ] Notifications
- [ ] Comment tweet
- [ ] Logout
- [ ] Like comment
- [ ] Delete comment
- [ ] Follow
- [ ] Forget the password
- [ ] Tweet with image

## Frontend

- Typescript
- React
